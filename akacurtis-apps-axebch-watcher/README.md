# AxeBCH ATH Watcher

Discord webhook notifications when mining workers hit new all-time high shares!

## Features

- ✅ Web-based configuration interface
- ✅ Real-time status monitoring
- ✅ Beautiful Discord embeds with progress bars
- ✅ Multi-worker support
- ✅ Persistent state across restarts

## Configuration

1. Open the app after installation
2. Paste your Discord webhook URL
3. Adjust poll interval (optional, default: 15 seconds)
4. Click Save

## Discord Webhook Setup

1. Discord → Server Settings → Integrations → Webhooks
2. Create New Webhook
3. Copy the webhook URL
4. Paste into the app configuration
